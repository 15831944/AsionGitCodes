課程索引
在這個課程上介紹的知識,可能會有錯誤和問題。但我只想盡我的全力去製作一個針對初學者的課程，讓他們在學習感到輕鬆和簡單。如果你對於學習OpenGL很認真，我建議你購買OpenGL Red Book(ISBN 0-201-46138-2)和OpenGL Blue Book，它們是至今為止最好的參考書。另一本我要推薦的書是OpenGL Superbible,當然你也可以選擇其它的。當你在這個網站上學習時，你應該多瀏覽一下其它的站點，如OpenGL.org. 你也可以瀏覽一些我提供的鏈接，大部分站點都是由一些非常優秀的編程人員創建的，它們很多都比我出色。我希望你能喜歡我提供給你的幫助，並且希望看到在不久的將來你能創建出你自己的程序。

最後一件重要的事是我的基礎代碼是在1997年寫的，如果有什麼改動的話，請留意我的網站：
http://nehe.gamedev.net/default.asp
創建一個OpenGL窗口:
在這個教程裡,我將教你在Windows環境中創建OpenGL程序.它將顯示一個空的OpenGL窗口,可以在窗口和全屏模式下切換,按ESC退出.它是我們以後應用程序的框架.

理解OpenGL如何工作非常重要，你可以在教程的末尾下載源程序，但我強烈建議你至少讀一遍教程，然後再開始編程.
你的第一個多邊形:
在第一個教程的基礎上，我們添加了一個三角形和一個四邊形。也許你認為這很簡單，但你已經邁出了一大步，要知道任何在OpenGL中繪製的模型都會被分解為這兩種簡單的圖形。

讀完了這一課，你會學到如何在空間放置模型，並且會知道深度緩存的概念。
添加顏色:
作為第二課的擴展，我將叫你如何使用顏色。你將理解兩種著色模式，在左圖中，三角形用的是光滑著色，四邊形用的是平面著色。
旋轉:
在這一課裡，我將教會你如何旋轉三角形和四邊形。左圖中的三角形沿Y軸旋轉，四邊形沿著X軸旋轉。
3D空間:
我們使用多邊形和四邊形創建3D物體，在這一課裡，我們把三角形變為立體的金子塔形狀，把四邊形變為立方體。
紋理映射:
在這一課裡，我將教會你如何把紋理映射到立方體的六個面。
光照和鍵盤控制:
在這一課裡，我們將添加光照和鍵盤控制，它讓程序看起來更美觀。
混合:
在這一課裡，我們在紋理的基礎上加上了混合，它看起具有透明的效果，當然解釋它不是那麼容易，當希望你喜歡它。
3D空間中移動圖像:
你想知道如何在3D空間中移動物體，你想知道如何在屏幕上繪製一個圖像，而讓圖像的背景色變為透明，你希望有一個簡單的動畫。這一課將教會你所有的一切。前面的課程涵蓋了基礎的OpenGL，每一課都是在前一課的基礎上創建的。這一課是前面幾課知識的綜合，當你學習這課時，請確保你已經掌握了前面幾課的知識。
加載3D世界，並在其中漫遊:
在這一課中，你將學會如何加載3D世界，並在3D世界中漫遊。這一課使用第一課的代碼，當然在課程說明中我只介紹改變了代碼。
飄動的旗幟:
這一課從第六課的代碼開始，創建一個飄動的旗幟。我相信在這課結束的時候，你可以掌握紋理映射和混合操作。
顯示列表:
想知道如何加速你的OpenGL程序麼？這一課將告訴你如何使用OpenGL的顯示列表，它通過預編譯OpenGL命令來加速你的程序，並可以為你省去很多重複的代碼。
圖像字體:
這一課我們將創建一些基於2D圖像的字體，它們可以縮放，但不能旋轉，並且總是面向前方，但作為基本的顯示來說，我想已經夠了。
圖形字體:
在一課我們將教你繪製3D的圖形字體，它們可像一般的3D模型一樣被變換。
圖形字體的紋理映射:
這一課，我們將在上一課的基礎上創建帶有紋理的字體，它真的很簡單。
霧:
這一課是基於第7課的代碼的，你將學會三種不同的霧的計算方法，以及怎樣設置霧的顏色和霧的範圍。
2D 圖像文字:
在這一課中，你將學會如何使用四邊形紋理貼圖把文字顯示在屏幕上。你將學會如何把256個不同的文字從一個256x256的紋理圖像中分別提取出來，並為每一個文字創建一個顯示列表，接著創建一個輸出函數來創建任意你希望的文字。
二次幾何體:
利用二次幾何體，你可以很容易的創建球，圓盤，圓柱和圓錐。
粒子系統:
你是否希望創建爆炸，噴泉，流星之類的效果。這一課將告訴你如何創建一個簡單的例子系統，並用它來創建一種噴射的效果。
蒙板:
到目前為止你已經學會如何使用alpha混合，把一個透明物體渲染到屏幕上了，但有的使用它看起來並不是那麼的復合你的心意。使用蒙板技術，將會按照你蒙板的位置精確的繪製。
線，反走樣，計時，正投影和簡單的聲音:
這是我第一個大的教程，它將包括線，反走樣，計時，正投影和簡單的聲音。希望這一課中的東西能讓每個人感到高興。
凹凸映射，多重紋理擴展:
這是一課高級教程，請確信你對基本知識已經非常瞭解了。這一課是基於第六課的代碼的，它將建立一個非常酷的立體紋理效果。
球面映射:
這一個將教會你如何把環境紋理包裹在你的3D模型上，讓它看起來像反射了周圍的場景一樣。
擴展，剪裁和TGA圖像文件的加載:
在這一課裡，你將學會如何讀取你顯卡支持的OpenGL的擴展，並在你指定的剪裁區域把它顯示出來。
變形和從文件中加載3D物體:
在這一課中，你將學會如何從文件加載3D模型，並且平滑的從一個模型變換為另一個模型。
剪裁平面，蒙板緩存和反射:
在這一課中你將學會如何創建鏡面顯示效果，它使用剪裁平面，蒙板緩存等OpenGL中一些高級的技巧。
影子:
這是一個高級的主題，請確信你已經熟練的掌握了基本的OpenGL，並熟悉蒙板緩存。當然它會給你留下深刻的印象的。
貝塞爾曲面:
這是一課關於數學運算的，沒有別的內容了。來，有信心就看看它吧。
Blitter 函數:
類似於DirectDraw的blit函數，過時的技術，我們有實現了它。它非常的簡單，就是把一塊紋理貼到另一塊紋理上。
碰撞檢測:
這是一課激動的教程，你也許等待它多時了。你將學會碰撞剪裁，物理模擬太多的東西，慢慢期待吧。
模型加載:
你知道大名鼎鼎的Milkshape3D建模軟件麼，我們將加載它的模型，當然你可以加載任何你認為不錯的模型。
拾取, Alpha混合, Alpha測試, 排序:
這又是一個小遊戲，交給的東西會很多，慢慢體會吧
加載壓縮和未壓縮的TGA文件:
在這一課裡，你將學會如何加載壓縮和為壓縮的TGA文件，由於它使用RLE壓縮，所以非常的簡單，你能很快地熟悉它的。
從高度圖生成地形:
這一課將教會你如何從一個2D的灰度圖創建地形
在OpenGL中播放AVI:
在OpenGL中如何播放AVI呢？利用Windows的API把每一幀作為紋理綁定到OpenGL中，雖然很慢，但它的效果不錯。你可以試試。
放射模糊和渲染到紋理:
如何實現放射狀的濾鏡效果呢，看上去很難，其實很簡單。把渲染得圖像作為紋理提取出來，在利用OpenGL本身自帶的紋理過濾，就能實現這種效果，不信，你試試。
卡通映射:
什麼是卡通了，一個輪廓加上少量的幾種顏色。使用一維紋理映射，你也可以實現這種效果。
從資源文件中載入圖像:
如何把圖像數據保存到*.exe程序中，使用Windows的資源文件吧，它既簡單又實用。
物理模擬簡介:
還記得高中的物理吧，直線運動，自由落體運動，彈簧。在這一課裡，我們將創造這一切。
繩子的模擬:
怎樣模擬一根繩子呢，把它想像成一個個緊密排列的點，怎麼樣有了思路了吧，在這一課你你將學會怎樣建模，簡單吧，你能模擬更多。
體積霧氣
把霧坐標綁定到頂點，你可以在霧中漫遊，體驗一下吧。
多重視口
畫中畫效果，很酷吧。使用視口它變得很簡單，但渲染四次可會大大降低你的顯示速度哦：）
在OpenGL中使用FreeType庫
使用FreeType庫可以創建非常好看的反走樣的字體，記住暴雪公司就是使用這個庫的，就是那個做魔獸世界的。嘗試一下吧，我只告訴你了基本的使用方式，你可以走的更遠。
3D 光暈
當鏡頭對準太陽的時候就會出現這種效果，模擬它非常的簡單，一點數學和紋理貼圖就夠了。好好看看吧。
頂點緩存
你想更快地繪製麼？直接操作顯卡吧，這可是當前的圖形技術，不要猶豫，我帶你入門。接下來，你自己向前走吧。
全屏反走樣
當今顯卡的強大功能，你幾乎什麼都不用做，只需要在創建窗口的時候該一個數據。看看吧，驅動程序為你做完了一切。
CG 頂點腳本
nVidio的面向GPU的C語言，如果你相信它就好好學學吧，同樣這裡也只是個入門。記住，類似的語言還有微軟的HLSL,OpenGL的GLSL,ATI的shaderMonker。不要選錯哦:)
軌跡球實現的鼠標旋轉
使用鼠標旋轉物體,很簡單也有很多實現方法,這裡我們教會你模擬軌跡球來實現它.
